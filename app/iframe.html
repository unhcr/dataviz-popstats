Iframe example



The visualisation is responsive by 'height'. The width to height ratio for the container should be 2:1 - in the example above, the div has width: 800px, height: 400px.

<div style="width: 800px; height: 400px"> <iframe src="http://data.unhcr.org/nepal/index.html" height="100%" width="100%" style="border:none;"></iframe> </div>
